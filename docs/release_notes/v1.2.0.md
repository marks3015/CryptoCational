# What's New in v1.2.0

## ✨ New Features

- **RSA-OAEP from Scratch**: Full pure-Python implementation of RSA-2048 with OAEP padding (SHA3-256 / MGF1).
- **Key Generation**: Miller-Rabin probabilistic primality test (`k=64` rounds) generates secure 2048-bit key pairs.
- **Asymmetric Encryption/Decryption**: Encrypt messages up to ~190 bytes with a public key and decrypt with the matching private key.
- **Digital Signatures**: Sign and verify files using OAEP-encoded SHA3-256 hashes with a PEM-like signed document format.
- **RSA-OAEP Page**: Dedicated UI with five sub-pages (Home, Key Generation, Encrypt, Decrypt, Sign/Verify), including a background `QThread` worker for non-blocking prime generation.
- **Agent Guide**: New `AGENTS.md` file to help future contributors and AI agents understand the project structure and conventions.

## 🔧 Technical Highlights

- Zero external cryptographic libraries — all RSA, OAEP, Miller-Rabin, and signature primitives implemented manually.
- JSON/Base64 key serialization for easy copy/paste and sharing.
- `secrets` module used for all random choices (primes, OAEP seeds), ensuring cryptographically strong entropy.
- Bilingual UI (English/Portuguese) extended to the new RSA-OAEP page.
- Round-trip tests verified for encryption/decryption and signature verification.
