# References / Referências

## English
This project is based on established cryptographic theories and statistical frequency analysis. Below are the primary resources used for benchmarking and pedagogical reference.

- [**Letter Frequency (Wikipedia)**](https://pt.wikipedia.org/wiki/Frequ%C3%AAncia_de_letras): Statistical source for letter distribution across different languages.
