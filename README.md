# CryptoCational

## English
**CryptoCational** is a modern, interactive educational tool designed for studying and applying cryptographic algorithms. In this **initial version**, the application focuses specifically on the Vigenère cipher, providing a robust foundation for future expansion. Built with **PySide6**, it emphasizes a minimalist user experience (UX) and rigorous statistical honesty.

### 🎯 Vision & Scalability
The project is designed to scale dynamically. While **Version 1.0** concentrates on the Vigenère implementation, future releases will incorporate a wide array of other famous cryptographic and cryptanalysis methods. Our goal is to maintain a practical and intuitive environment that simplifies complex encryption techniques for students and enthusiasts.

![Logo](assets/images/logo.png)

### 🚀 Key Features
- **Format-Preserving Encryption**: Encrypt texts while maintaining spaces, punctuation, casing, and line breaks.
- **Robust File I/O**: Direct import of `.txt` files for processing large volumes of data.
- **Statistical Cryptanalysis**: Integrated tools for breaking ciphers using Index of Coincidence (IC) and Chi-Squared ($\chi^2$) frequency analysis.
- **Modern Dark-Mode UI**: A sleek, responsive interface designed for focus and productivity.
- **Bilingual Core**: Full interface and documentation support for both English and Portuguese.

## Português
**CryptoCational** é uma ferramenta educacional moderna e interativa projetada para o estudo e aplicação de algoritmos criptográficos. Nesta **versão inicial**, a aplicação foca especificamente na cifra de Vigenère, fornecendo uma base robusta para expansão futura. Construída com **PySide6**, enfatiza uma experiência de usuário (UX) minimalista e rigor estatístico.

### 🎯 Visão e Escalabilidade
O projeto foi desenvolvido para escalar dinamicamente. Embora a **Versão 1.0** se concentre na implementação de Vigenère, os lançamentos futuros incorporarão uma ampla gama de outros métodos famosos de criptografia e criptoanálise. Nosso objetivo é manter um ambiente prático e intuitivo que facilite o entendimento de técnicas complexas para estudantes e entusiastas.

### 🚀 Principais Funcionalidades
- **Criptografia com Preservação de Formato**: Criptografe textos mantendo espaços, pontuação, maiúsculas/minúsculas e quebras de linha.
- **I/O de Arquivos Robusto**: Importação direta de arquivos `.txt` para processamento de grandes volumes de dados.
- **Criptoanálise Estatística**: Ferramentas integradas para quebrar cifras usando Índice de Coincidência (IC) e análise de frequência Qui-Quadrado ($\chi^2$).
- **UI Moderna em Dark-Mode**: Uma interface elegante e responsiva projetada para foco e produtividade.
- **Core Bilíngue**: Suporte total de interface e documentação para Inglês e Português.
